# Explore with me
## Описание
Explore with me - проект, позволяющий создавать и делиться различными событиями с другими людьми.
А также помогает другим пользователям находить уже созданные события и принимать в них участие.

# Комментарии
## Описание
Новая фича позволяет оставлять комментарии к событиям </br>
Ссылка на pull request: https://github.com/stepReez/java-explore-with-me/pull/5
## Endpoints
### Приватные
| Method   | URL                                  | Description                                                    |
|----------|--------------------------------------|----------------------------------------------------------------|
| `POST`   | `/users/{userId}/comments/{eventId}` | Добавление комментария текущем пользователем заданному событию |
| `PATCH`  | `/users/{userId}/comments/{comId}`   | Изменение своего комментария текущем пользователем             |
| `GET`    | `/users/{userId}/comments`           | Получение комментариев, отправленных текущем пользователем     |
| `GET`    | `/users/{userId}/comments/{comId}`   | Получение определенного комментария текущем пользователем      |
| `DELETE` | `/users/{userId}/comments/{comId}`   | Удаление определенного комментария текущем пользователем       |

### Административные
| Method   | URL                       | Description                          | 
|----------|---------------------------|--------------------------------------|
| `DELETE` | `/admin/comments/{comID}` | Удаление комментария администратором |

### Публичные
Теперь эндпоинт `GET` `/events/{id}` возвращает событие с комментариями
